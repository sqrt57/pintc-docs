# pintc-docs

Design documentation, specifications, and working notes for the Pint compiler.

Pint is a minimal systems language targeting Win32 native binary (x86).

- **Language spec:** [pint-spec-v0.6.md](pint-spec-v0.6.md)
- **Compiler source:** [pintc-cs](https://github.com/sqrt57/pintc-cs) (C# implementation)
